HTML IN SHORT

HTML is markup language for describing web documents.

HTML stands for Hyper Text Markup Language

HTML documents are described by HTML tags.

HTML Elements

There are two types of HTML elemnts

1.Block level elements

A block level element always start from new line and takes up the full width available.

Examples: <div>,<p>,<form>,<h1> to <h6>

2.Inline elements

An inline element does not start on new line and takes up only as width as needed.

Examples: <span>,<a>,<img>

HTML Tags

HTML tags mostly come in pairs.
The first tag is start tag and second one is called end tag.
HTML tags are keywords surrounded by angular brackets.

<tagname>content</tagname>

Some widely used tags:

1.HTML headings

<h1>This is heading</h1> <h2>This is heading</h2> <h3>This is heading</h3> <h4>This is heading</h4> <h5>This is heading</h5> <h6>This is heading</h6> <h1> indicates most important heading <h6> indicates least important heading

2.HTML paragraphs

<p>This is paragraph</p> <p>This is another paragraph</p>

3.HTML Links

HTML links are defined by <a> tags. <a href="www.google.com">Click here for Google </a>

Clicking on this will redirect to google home page.

4.HTML Images

HTML images are defined by <img> tags. <img src=”” alt=”My home” width=”100px” height=”100px”> ‘src’ indicates path of the image . ‘alt’ indicates text to be shown when image is not loaded properly. width , height provides dimensions of the image. <p>Clicking on this will redirect to google home page.</p>

5.HTML CSS

The css for an element can be declared as inline to the element using ‘style’ attribute.

<h1 style=”color:blue;”>HTML5 POWER UNLEASHED</h1>

The above mentioned example changes the color of heading to blue.
Note: Inline CSS is not an advisable practice for implementing CSS.

HTML Attributes

HTML elements can have attribute. They provide additional information about an element. Attributes are specified in start tags and always come in key-value pairs.

Some widely used attributes

1.lang attribute

The document language can be declared in <html> tag. It is important for accessibility applications and search engines. <html lang=”en”>

2.title attribute

We can declare title attribute to elements such as <p>,<div>,<h1> etc. On hovering on these tags, a small header is displayed.

3.href attribute

This attribute is used in <a> tag mainly for link address, to where page should be navigated. <a href="linkPath">Click here to go to link</a>

4.title attribute

We can declare title attribute to elements such as <p>,<div>,<h1> etc. On hovering on these tags, a small header is displayed.

5.size attribute

On observing <img> tag, <img src=”” alt=”” width=”” height=””>. All the details of image are mentioned through attributes ‘src’ , ‘alt’ , ‘width’ and ‘height’. <p>'src' attribute denotes path of image</p> <p>'alt' attribute denotes text to be shown when image fails to load</p> <p>'width' , 'height' denotes dimensions of the image.</p>
